# Healthcare Claims Intelligence Lakehouse

This project demonstrates a real-world healthcare claims data engineering pipeline using Python, PySpark-style transformations, validation rules, and Bronze/Silver/Gold lakehouse architecture.

## Objective

Build a healthcare claims pipeline that ingests raw claim records, validates claim quality, removes invalid records, creates curated claim datasets, and produces analytics-ready claim metrics.

## Architecture

Raw Claims Data
→ Bronze Layer
→ Silver Validation Layer
→ Gold Analytics Layer

## Tech Stack

- Python
- PySpark
- SQL concepts
- Parquet
- Data quality checks
- Bronze/Silver/Gold architecture
- Git/GitHub
